# ✈️ MakeMyTrip Clone – Full-Stack Travel Booking System

## 📌 Overview

This project is a **full-stack travel booking application** inspired by MakeMyTrip. It allows users to search and manage travel bookings through a responsive web interface with a robust backend system.

The application demonstrates **end-to-end development**, including frontend UI, backend APIs, database integration, and deployment.

---

## 🚀 Tech Stack

### 🧠 Backend

* Java
* Spring Boot
* REST APIs
* Maven

### 🌐 Frontend

* Next.js (React Framework)
* TypeScript
* HTML, CSS

### 🗄️ Database

* MongoDB

### ⚙️ Tools & Platforms

* Git & GitHub
* Postman (API Testing)
* Docker (Containerization)
* Render (Deployment)
* IntelliJ IDEA (Backend Development)
* VS Code (Frontend Development)

---

## ✨ Features

* 🔐 User Authentication (Login/Register)
* 🔍 Search and browse travel options
* 📅 Booking workflow management
* 🔄 Real-time data handling with API integration
* 📱 Responsive UI using modern frontend technologies
* ☁️ Deployment using Render with Docker support
